## Keybind
- Ranger
  * `<Leader> + n` or `<F5>` Open Current File
  * `<Leader> + c` Open Current Directory
  * `<Leader> + f` Open Project Root Directory

- `<F7>` Toggle Tagbar
- Fzf
  * `<F2>` Open Files
  * `<F3>` Open Buffers
  * `<F4>` Open Buffer Lines
